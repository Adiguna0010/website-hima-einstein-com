# HIMA EINSTEIN.COM - Website Resmi Himpunan Elektronika & Instrumentasi
## Kabinet Phótisma 💡

Selamat datang di repository resmi untuk pengembangan **Website HIMA EINSTEIN.COM**, Himpunan Mahasiswa Elektronika Instrumentasi Politeknik Teknologi Nuklir Indonesia. Website ini dirancang secara profesional, elegan, dan berkelas internasional sebagai pusat informasi, media publikasi, dan identitas digital Himpunan.

---

## 🌟 Tentang Kabinet Phótisma
Nama **Phótisma** (dari bahasa Yunani: *φώτισμα* yang berarti "Cahaya" atau "Penerangan") melambangkan visi kabinet ini untuk membawa terang, inspirasi, dan inovasi bagi seluruh anggota himpunan, sivitas akademika, dan masyarakat luas melalui teknologi elektronika instrumentasi nuklir.

---

## 📂 Struktur Repositori
Repositori ini diawali dengan struktur berikut:
- `Media Einsten/` - Folder penyimpanan media publikasi Himpunan.
  - `Media Divisi/` - Dokumentasi dan aset per divisi.
    - `Divisi Pengembangan Mahasiswa/`
    - `Divisi Riset Dan Teknologi/`
  - `Media Umum/` - Dokumentasi kegiatan umum Himpunan.

---

## 🚀 Rencana Pengembangan Website
Website ini akan dikembangkan dengan desain modern, dinamis, dan responsif. Beberapa fitur utama yang direncanakan:
1. **Landing Page**: Pengenalan Himpunan, Visi & Misi Kabinet Phótisma, dan Sambutan Ketua Himpunan.
2. **Profil Pengurus**: Informasi pengurus Himpunan Elektronika Instrumentasi.
3. **Divisi & Program Kerja**: Halaman khusus yang memaparkan kegiatan tiap divisi.
4. **Blog / Berita & Kegiatan**: Informasi terbaru seputar kegiatan Himpunan dan artikel sains/teknologi.
5. **Galeri Media (Media Einstein)**: Integrasi media dokumentasi kegiatan dari folder `Media Einsten`.

---

## 🤝 Kontribusi & Kolaborasi
Repositori ini dikonfigurasi untuk kolaborasi tim. Silakan lakukan langkah berikut untuk berkontribusi:
1. Clone repositori ini ke komputer lokal Anda.
2. Buat branch baru untuk fitur yang ingin Anda kerjakan:
   ```bash
   git checkout -b fitur/nama-fitur
   ```
3. Lakukan commit dengan pesan commit yang jelas dan deskriptif.
4. Push branch Anda dan buat Pull Request (PR) ke branch utama (`main`).
